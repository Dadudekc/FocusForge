## 🛠️ Development

### Prerequisites

- Python 3.8+
- PyQt5
- SQLite3

### Setup

1. Clone the repository
2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running

```bash
python main.py
```
